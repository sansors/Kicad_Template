# Kicad_Template
My own template for kicad

A template is support and model for create a daughter board of a primary setup.
Typically, it's a shield for a arduino or raspberry.

If you want to use a template, or made your own's, please read the documentation [HERE](https://docs.kicad-pcb.org/5.0.2/es/kicad/kicad.html#project_templates)
